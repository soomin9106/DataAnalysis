# DataAnalysis
여러 분석 기법들을 통한 데이터 분석 실습 자료 입니다.

#시계열 분석 절차
1. 시간그래프 그리기
2. 추세와 계절성을 제거하기
3. 잔차를 예측하기
4. 잔차에 대한 모델 적합하기
5. 예측된 잔차에 추세와 계절성을 더하여 미래를 예측하기

- 자기회귀모형
AR 모형
p시점 전의 자료가 현재 자료에 영향을 주는 모형
(p는 시점)
- 이동평균모형
MA 모형
유한한 개수의 백색잡음의 결합이므로 언제나 정상성을 만족
백색잡음과정은 오차항을 의미
- 자기회귀누적이동평균모형
ARIMA 모형
ARIMA 모형은 비정상시계열 모형
